## 今日触れたこと
- gitはデータをスナップショットとして記録している
- コミットを辿ることで以前の状態に戻すことが出来る
- git add で特定のファイルをステージに追加
- git commit でステージに追加したファイルをリポジトリに記録する　これがコミット。
- コミットした時、リポジトリには圧縮ファイル（正確な表現ではないが）、ツリー、コミットの三つのファイルが保存される。
