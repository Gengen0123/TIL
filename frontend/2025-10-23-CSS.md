# グリッドレイアウトの初歩

ここでもコンテナという言葉が出てくるのかという気持ち
常に正方形にするには？

```
<!doctype html>
<html lang="ja">

<head>
    <meta charset="utf-8" />
    <title>Grid 最小サンプル</title>
    <style>
        /* これが“グリッドコンテナ” */
        .grid {
            display: grid;
            /* グリッドを有効化 */
            grid-template-columns: 1fr 1fr 1fr;
            /* 3列レイアウト（frは“比率”） */
            gap: 12px;
            /* すき間 */
            padding: 12px;
            background: #f4f4f4;
        }

        /* 中の箱（グリッドアイテム） */
        .item {
            background: white;
            border: 1px solid #ddd;
            padding: 16px;
            font: 14px/1.4 system-ui, sans-serif;
            border-radius: 6px;
        }

        /* アイテムを横に2列ぶんまたぐ例 */
        .span-2 {@
            grid-column: span 2;
        }

        /* 画面幅が狭いときは“自動折返し”のレスポンシブ版に切替え */
        @media (max-width: 640px) {
            .grid {
                grid-template-columns: repeat(auto-fit, minmax(160px, 1fr));
                /* 幅160pxを下回らないようにカードを自動で段組 */
            }
        }
    </style>
</head>

<body>

    <div class="grid">
        <div class="item">① 普通のアイテム</div>
        <div class="item">② 普通のアイテム</div>
        <div class="item">③ 普通のアイテム</div>

        <div class="item span-2">④ これは横に2列ぶん広がる（grid-column: span 2）</div>
        <div class="item">⑤ 普通のアイテム</div>

        <div class="item">⑥ 普通のアイテム</div>
        <div class="item">⑦ 普通のアイテム</div>
        <div class="item">⑧ 普通のアイテム</div>
    </div>

</body>

</html>
```
