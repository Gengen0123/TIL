# 基礎的なJSを使用した動的サイトの作り方

## ボタンを押すと　こんにちは　と表示させる

```
 <p>Hello World!</p>
    <p id="click">ボタンをクリックしてね</p>
    <button onclick="clickNow()">ボタン</button>

    function clickNow() {
            var click = document.querySelector('#click');
            click.textContent = "こんにちは！";
        }
```
## テキストボックスに入力した値を表示する

```
<p id="click2">テキストを入力</p>
    <p>
        <input type="text" id="text1">
        <button onclick="clickNow2()">決定</button>

    </p>

    function clickNow2() {
            var t1 = document.getElementById('text1');
            var value = t1.value;
            var click2 = document.querySelector('#click2');
            click2.textContent = value;
        }
```
